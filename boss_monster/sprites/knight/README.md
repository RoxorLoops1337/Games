# Knight sprite sheets

Drop the knight sprite-sheet PNG(s) here (alpha/transparency preserved when
uploaded via GitHub's "Add file → Upload files", not as a chat image).

When you add them, please also note (in a comment or a small atlas .json/.txt):
- frame size (e.g. 64x64) and grid (cols x rows)
- which rows/ranges are which animation (idle / walk / attack / hurt / death)
- frame count + FPS per animation
- one sheet per animation, or one combined sheet
