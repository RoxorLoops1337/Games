# Champion knight — the big empowered hero (mini-boss)

Sprite for the CHAMPION raider (the powered-up hero mini-boss). For now: the
WALK animation. Heroes march rightward, so the right-facing row is what gets used.

Upload via GitHub's "Add file → Upload files" (not as a chat image) so
transparency is preserved. Export WITHOUT a baked shadow (the engine draws it).

Any filenames are fine — I'll normalise them. To save me guessing, note:
- frame size (e.g. 64x64 or 192x192) and grid (cols x rows)
- which rows are which facing (usually up / left / down / right)
- walk frame count + FPS
- one sheet, or one PNG per frame — either works.

He should read clearly BIGGER than a normal hero (he's a mini-boss). Same footing
(feet at the bottom of the frame / aligned across frames) so the walk doesn't jitter.
